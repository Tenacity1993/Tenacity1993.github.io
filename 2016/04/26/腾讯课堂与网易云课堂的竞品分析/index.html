腾讯课堂与网易云课堂的竞品分析

发表于 2016-04-26

一. 背景

在线教育自提出以来就一直受到广泛关注，近年来更是迅猛发展，作为一个新生市场，未来的发展空间巨大。在线教育打破了传统意义上的学习方式，突破了时间和空间上的限制，开辟了随时学习，移动学习的新模式，提高了学习的效率；另一方面，借助于互联网这个平台，可以跨越因地域等方面造成的教育资源不平等分配，使教育资源共享化，降低了学习的门槛，尤其是使得足不出户学习国外名校课程成为了可能。随着在线教育的发展，不仅越来越多的人认可在线教育，在线教育的快速发展也获得了互联网巨头和资本市场的青睐，越来越多的公司加入到在校教育的行列中来。根据在线教育垂直网站芥末堆的统计显示，我国教育行业在2015 年对外公布的投融资数量为199起，总融资金额达到了20.3亿美元，相较于2014年，投融资数量增长了36.6%，总融资额则增长了160%。20.3亿元的融资额，也让中国在线教育首次超过美国，成为全球在线教育最热的投资市场。下面选取两个在线教育产品进行分析，分别是腾讯课堂和网易云课堂。

二. 分析对象

本文选择腾讯课堂和网易云课堂两个产品进行竞品分析。
腾讯课堂是腾讯公司推出的专业在线教育平台，聚合了优质机构和海量课程资源，2014年4月上线。腾讯课堂凭借QQ的优势，实现了在线即时互动教学。依托强大的用户群，打破地域限制，为学生提供学习的平台，也为优秀的机构和教师提供了一个展示的平台。

网易云课堂是网易公司倾力打造的在线实用技能学习平台，2012年12月底上线（比腾讯课堂早），主要为学习者提供海量优质的课程，为学习者提供一站式的学习服务。从用户生活、职业、娱乐等多个维度，为用户打造实用学习平台。

三. 用户分析

对于这两个在线教育平台而言，目标用户群体大致是一样的，大家的竞争点是用户的时间。对于学生而言，是那些认可在线教育，希望通过在线教育平台学习知识、技能、充实、完善自己的人。对于机构而言，则是希望顺应互联网发展的潮流，结合线上线下平台，拓展自己的教育途径，宣传自己的教育品牌，在提供教育输出的同时获得利润的机构。

五. 功能分析

1. 从产品标语来看：

腾讯课堂——专业的在线教育平台，优势是课程资源丰富，通过与线下教育机构的合作，为用户提供良好的学习体验。

网易云课堂——领先的实用技能学习平台，可以看出云课堂将焦点放在实用技能上，更多的以职业为导向，面向求职者、面向待求职大学生，但是同时各年龄段的教育也都涵盖。

2. 从课程内容和数量来看：

两个产品的课程设置大致相同，首页显示标签也大致类型相同。两个产品都把IT·互联网教育放在了标签的首位，语言学习放在了第三位。云课堂第二位是职业技能，腾讯课堂则是设计·创作。内容上涵盖了学生各阶段教育，职业教育，生活教育的各个层面，内容丰富。

从数量上来看，腾讯课堂的课程数量多于网易云课堂。截至2015年，腾讯课堂共推出7.5万门精品课程，为千万学员提供优质教学服务。网易云课堂课程数量超10000，课时总数超100000，其中付费课程少于三分之一。

3. 从教学机构来看：

在IT互联网教育下，网易云课堂主要由高校、网易杭州研究院和外部机构共同组成，其中高校和网易杭研院课程占了不小的比重，腾讯课堂则全部由入驻机构组成。其他标签下的课程云课堂有高校参与，而腾讯课堂则都是外部机构来完成。截至2015年，腾讯课堂入驻机构总数达到1.3万家。

所以在机构组成上，由于有高校参与，整体看来大众对网易云课堂的认可度较高。而腾讯课堂入驻机构较多，机构水平层次各不相同，怎样筛选孵化优质机构也是腾讯课堂日后要努力解决的问题。

4. 从二者特有功能来看：

网易云课堂

  • 微专业教育
    云课堂设有产品设计、前台开发、后台开发、技术支撑、艺术、其他（会计主管、税务会计主管、高级人力资源专员）六大微专业。为学习者提供跟随知名企业的一线资深工程师、设计师以及行业知名专家学习的机会，旨在让学习者能够系统掌握工作方法和技巧，获得全新的职业提升。（课程需要付费）
  • 提供三大课程体系
    • 大学计算机专业，汇集多所大学广受好评的计算机老师，且课程由教师自行制作。（课程免费）
    • 互联网职业技能，由行业内多个领域知名专家和机构共同完成，内容全面。（本部分内容与微专业内容有重叠，且需付费）
    • 金融专业，本部分由清华大学五道口金融学院旗下在线教育平台“紫荆教育”提供。（需付费）
  • 云课堂笔记功能
    学习者添加笔记时会自动保存视频的当前时间点，回顾笔记时就可观看当时视频，还可以对视频截图或上传本地图片保存到笔记当中，这使得用户可以更快捷方便的保存老师的板书和重要信息。同时学习者可将笔记公开，也可评论、收藏他人笔记，加强了用户间学习交流。不过在交流的实时性上，不如腾讯课堂便捷。
  • 进度管理与学习监督
    学习者学习过程中，支持自动\手动标记课时完成状态，或标记为“重要\有疑问”等，以便用户回顾和把控学习进度。另外，用户可设置课程的学习时间安排，云课堂会定期发送提醒通知用户。
  • 奋斗模式
    通过开课前提前支付直接申请证书一半的费用（一般是50元）加入奋斗模式，加入用户将享受全程论坛高亮显示以及作业截至邮件提醒服务。在奋斗期间可以申请退款，课程结束后，成绩通过可选择申请证书或者退款。
  • 证书体系
    电子成绩单免费，认证证书收费（电子+纸质）。成绩达到标准可以申请证书，如若未达到，可享受下次开课证书免单一次，且证书可以直接添加用户在linkedin、猎聘网、周伯通等网站的简历。
  • 结束课程有存档，只要课程不关闭可以随时观看。

    腾讯课堂

  • 与QQ绑定，可以从QQ面板中方便的选择课程，同时还有QQ弹窗提醒功能。
  • 腾讯课堂重点推出直播课程，支持在线即时互动教学，提供流畅、高音质的课程直播效果；支持PPT演示，屏幕分享等多样化的授课模式。同时支持游客模式。
  • 通过机构认证官方Q群，可以方便的与同学、老师交流。机构可以通过Q群建立课程表，便于用户发现课程。
  • 新版的QQ中，完善了家校师生群“作业”功能，引入了第三方题库，覆盖小学、初中、高中各门学科，教师布置作业时从题库中快速组题，学生在线提交后自动批阅。QQ家校·师生群有一键提醒功能，对于未能按时完成作业的学生集中提醒，并且还能一键提醒家长查阅。
  • 学团功能，为学生提供交流的论坛。
  • 付费课程支持试听，为用户提供了更多选择的空间，同时腾讯为用户提供大咖老师专属课程优惠券。
  • 设计就业挑战赛，报名相应付费课程的同学可以参加挑战赛，入围作品会在腾讯课堂首页展示，获奖者有机会参加企业招聘在线面试。

七. 产品优缺点对比

网易云课堂

优点

1.提供微专业体系，为特定专业人群提供了完整的学习计划，尤其是计算机课程计划，课程全部由高校老师主讲，体系完整且免费；
2.与高校合作，课程质量有保证；
3.奋斗模式可以督促学生提高学习效率，保证听课质量；
4.起步早、免费课程占比大、用户口碑较好；
5.笔记功能同时截取时间点，方便用户复习；
6.有证书认证体系，能够给用户带来学习上的成就感；
7.已经结束的课程有存档，可以随时观看；
8.每月显示本月最值得推荐的课，为学习者提供学习建议；
9.能够根据用户的浏览提供“猜你喜欢”课程分类，方便学习者选择；
10.设有部分直播课程，弥补了课程体系的不足；
11.部分课程获得高校认可，参与高校日常教学。

缺点

1.在线学习的自主性较大，对于部分学生而言，奋斗模式造成了一定的压迫性，可能会流失部分用户；
2.虽然设有讨论区，但是不能实现老师和学生实时的交流和沟通；
3.付费课程无法试听，用户无法确定自己对课程的感受以及课程的质量，会流失部分用户；
4.课程分类仍需改进，选课时可选标签较少，不能很快选到目标课程；
5.认证体系未能获得一定程度上的社会认可，实用性较低。

腾讯课堂

优点

1.与QQ深度绑定，方便学生，方便机构，沟通交流无障碍，提醒功能强大贴心，交互性好，移动学习体验较好；
2.设有微信公众号，能够及时提醒、及时督促；
3.设有学生论坛，方便用户交流分享；
4.课程直播回放给用户带来更好的体验；
5.课程选择标签多，用户定位课程较快；
6.课程体系完备，课程资源丰富。

缺点

1.课程有期限，部分直播课程结束后无法回放观看
2.由于学生层次不一样，在线直播教育无法适合所有人的水平，也没有提供差异化的课程；
3.腾讯课堂标签在qq中无法取消显示，且部分视频播放时有水印，也无法取消。
4.对视频进行反馈要单开页面，学习页面没有开始学习按钮，无法根据进度直接学习，需要手动点课程。
5.暂时还未打通微信学习体系。

八. 总结和探讨

通过分析我们发现，对于用户的需求而言，上述产品仍然不能很好的满足用户的需求。

  1. 产品没有意识到用户层次的差异性，没有很好的将学习者的能力与课程要求进行匹配，只是为所有人提供同样的课程，没有对课程进行难易层次区分。可以根据课程的难以程度分成：入门、进阶、高级。
  2. 课程资源区分不够明显，存在多门以“某某”入门等此类为标题的课程，使得用户对课程无法区分。
  3. 没有倍速播放功能，用户缺乏对课程的把握能力。
  4. 同类课程建议给出依据教师风格进行课程筛选的渠道，为用户提供更加明确的选择。

除此之外，两个产品还都面临着一些挑战：

1.课程平台虽然有着海量的课程资源，但是随着在线教育多年来的发展，各种领域的在线教育都发展迅速，存在着某些针对性较强的在线教育产品。如何在这种竞争中获得优势是两家公司都要考虑的问题。相关产品暂列举如下：

  • IT教育——慕课网、极客学院
  • 产品经理教育——起点学院
  • 高校教育——edx、course、学堂在线、腾讯公开课、中国大学MOOC、网易公开课（后两个为网易公司产品）
  • 外语教育——新东方在线
  • 留学教育——小站教育、无老师私塾

2.两个产品所在公司在教育领域各自都还有其他产品，网易的网易公开课、中国大学MOOC，腾讯的精品课等。如何在与以上产品的竞争合作中保持产品的优势也是需要思考的问题。
